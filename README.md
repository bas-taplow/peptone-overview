# Peptone infographic overview

A single self-contained page: the Peptone infographic sections with a style
toggle between **Original** (the site's own sections, on black) and
**Minimalist** (the same graphics re-skinned onto white, with a light/dark
toggle of its own). Live and animated — nothing is a screenshot.

Published at the Pages URL for this repo.

Only the built page lives here. It is generated output: the build scripts and
the saved source pages are kept in a separate private repo, and this file is
overwritten wholesale on each rebuild, so don't hand-edit it.
